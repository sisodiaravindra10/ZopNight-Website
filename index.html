faq

Questions we get a lot.

If yours isn't here, ask us directly.

Does ZopNight require agent installation or code changes? +

No agents, no sidecars, no Terraform changes. ZopNight connects through read-only cloud APIs. AWS, GCP, and Azure access is granted via IAM roles you provision once. Discovery starts in minutes, not days.

How quickly will I see findings? +

Most accounts see their first findings within 2–3 minutes of connecting. The initial scan covers all resources across regions. After that, ZopNight runs continuously. New waste is surfaced as it appears, not on a daily batch.

What does "one-click remediation" actually do? +

When you click Fix →, ZopNight calls the cloud provider API directly. Stopping an idle instance, deleting an orphaned volume, rightsizing a reservation. Every action is logged, reversible where the cloud allows it, and you can set approval workflows so nothing runs without a human sign-off.

Which clouds and services are supported? +

AWS, GCP, and Azure. Covering compute, storage, databases, networking, and Kubernetes across all regions. We add new service coverage every sprint. If you use something niche, tell us and it's usually on the roadmap within a quarter.

Is my cloud data safe? What does ZopNight read? +

ZopNight operates with least-privilege read access plus the specific write permissions for remediation actions you explicitly enable. We never read application data, secrets, or user content. Only resource metadata and billing data. All data is encrypted in transit and at rest. SOC 2 Type II report available on request.

Does it work alongside Terraform or other IaC tools? +

Yes. ZopNight detects IaC-managed resources and marks them separately so you know which fixes should go through your Terraform pipeline versus which can be actioned directly. For IaC resources, we can generate a diff-ready Terraform snippet rather than acting directly.

How is ZopNight priced? +

Pricing is based on the number of cloud resources managed. Not on savings found or actions taken. Most teams pay back the subscription cost in the first week. See full plans on the pricing page, or start free with the playground. No card required.
